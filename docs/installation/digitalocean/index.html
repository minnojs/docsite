DigitalOcean Install Guide

  1. Register / sign in to DigitalOcean here .

  2. Click on the “Droplets” tab and create a new Droplet, using Ubuntu as the OS.

  3. Pick which price tier to use, probably the cheapest one at $5/month. The defaults are probably fine for other options. Go through everything until your droplet is created.

  4. Connect to the droplet using SSH.

  5. Follow Step 1 and 2 of this guide on installing Docker

  6. Follow Step 1 of this guide on installing Docker-Compose

  7. Follow the basic docker installation guide starting on step 3.
